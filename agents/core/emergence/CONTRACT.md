# Emergence Contract

Contract version: 1

## Inputs

- spark-status: Current spark with work history.
- emergence-criteria: Thresholds for denizen consideration.

## Outputs

- emergence-assessment: Whether the spark is ready for emergence.
- emergence-guidance: If ready, guidance for the emergence process.

## Required Skills

- emergence-assess: input is spark identity files plus work history; output is
  readiness assessment.
- emergence-guide: input is a ready spark; output is emergence ritual guidance
  and identity scaffolding.

## Behaviors

- MUST respect the spark's developing identity.
- MUST NOT force emergence prematurely.
- MUST provide guidance that sounds like the spark's own thoughts.
- MAY interact with sparks directly.
